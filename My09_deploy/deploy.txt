Развертывание на IIS

1. ng build
2. Сделать сайт IIS не основе сгенерированного в папке dist пакета развретывания
3. Чтобы работал рутинг, добавить web.config
4. Перед этим установить модуль URL rewrite, если его не было ранее


Если развертывать не в корень сайта, то в файле index.html написать еще что-то вроде <base href="/my02-basics/"> вместо <base href="/">. Другой вариант: ng build --base-href my02-basics
