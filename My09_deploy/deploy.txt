Развертывание на IIS

1. ng build
2. Сделать сайт IIS на основе сгенерированного в папке dist пакета развретывания
3. Чтобы работал рутинг, добавить туда web.config (см.)
4. Перед этим установить модуль URL rewrite, если его не было ранее


Если развертывать не в корень сайта, то в файле index.html написать еще что-то вроде <base href="/my02-basics/"> вместо <base href="/">. Другой вариант для этого: ng build --base-href /my02-basics/
